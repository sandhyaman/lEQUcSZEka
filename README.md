# 项目简介 python846

本项目是一款基于Python语言的书籍销售管理系统，采用Django框架进行开发，并使用Mysql数据库进行数据存储。

## 技术栈

- Python
- Django框架
- Mysql数据库

## 功能模块

本项目包含以下功能模块：

1. 用户管理
2. 书籍管理
3. 订单管理
4. 营销活动
5. 数据报表

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 开发环境：Python 3.x，Django 2.x，Mysql 5.x
- 生产环境：与开发环境一致

## 部署步骤

1. 安装Python环境
2. 安装Django框架
3. 配置Mysql数据库
4. 运行项目源码

## 目录结构

```
project/
│
├── apps/ # 系统功能模块
├── db/ # 数据库相关文件
├── media/ # 静态文件
├── static/ # 静态资源文件
├── templates/ # 模板文件
├── manage.py # 项目管理脚本
└── requirements.txt # 项目依赖
```
